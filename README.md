# 안드로이드 - 아두이노 통신 프로젝트
## 개요
졸업 작품으로 제작한 안드로이드 - 아두이노 블루투스 통신 프로젝트입니다.

아두이노로 측정한 무게 데이터를 안드로이드 애플리케이션으로 전송하여 모바일 화면에서 데이터 확인이 가능합니다.

안드로이드 애플리케이션에서 아두이노를 제어하는 신호를 보내 아두이노를 제어할 수 있습니다.
